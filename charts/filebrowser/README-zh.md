File Browser 在指定目录内提供一个文件管理界面，可用于上传、删除、预览和编辑文件。这是一种“自建云”类型的软件，你只需将其安装在服务器上，指定一个路径，即可通过美观的 Web 界面访问你的文件。

## 文档

有关如何安装、配置以及参与本项目开发的文档，请访问 [filebrowser.org](https://filebrowser.org)。
