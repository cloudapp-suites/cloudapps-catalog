OpenClaw has been deployed!

  Chart version: {{ .Chart.Version }}
  App version:   {{ .Chart.AppVersion }}

1. Wait for the pod to be ready:

   kubectl get pods -n {{ .Release.Namespace }} -l app.kubernetes.io/name={{ include "openclaw.name" . }}

2. Access the gateway:

   kubectl port-forward -n {{ .Release.Namespace }} svc/{{ include "openclaw.fullname" . }} {{ .Values.service.port }}:18789

   Then open http://localhost:{{ .Values.service.port }} and enter your Gateway Token.

3. Approve the pairing request:

   kubectl exec -n {{ .Release.Namespace }} deployment/{{ include "openclaw.fullname" . }} -c main -- node dist/index.js devices list
   kubectl exec -n {{ .Release.Namespace }} deployment/{{ include "openclaw.fullname" . }} -c main -- node dist/index.js devices approve <REQUEST_ID>
