## 快速入门

完整入门指南请查看 https://nodered.org/docs/getting-started/。

1. `sudo npm install -g --unsafe-perm node-red`
2. `node-red`
3. 打开 <http://localhost:1880>

## 开发者指南

如需运行最新的 Git 代码，请按照以下步骤操作：

1. 克隆代码：

        git clone https://github.com/node-red/node-red.git
        cd node-red

2. 安装 Node-RED 依赖项：

        npm install

3. 构建代码：

        npm run build

4. 运行：

        npm start
