# Vane


Vane 是一款**以隐私为核心的人工智能问答引擎**，完全运行于您自己的硬件之上。它融合了海量互联网知识，并支持**本地大语言模型（Ollama）** 以及云服务提供商（OpenAI、Claude、Groq），在确保您的搜索行为完全私密的同时，提供准确且**附带引用来源**的答案。


想进一步了解其架构与工作原理？可在此阅读详细说明：[此处](https://github.com/ItzCrazyKns/Vane/tree/master/docs/architecture/README.md)。

## ✨ 主要特性

🤖 **全面支持主流 AI 服务商** —— 可通过 Ollama 使用本地大语言模型，也可连接 OpenAI、Anthropic Claude、Google Gemini、Groq 等云端服务；您可根据实际需求自由组合不同模型。

⚡ **智能搜索模式** —— 需要快速响应时选择「极速模式」，日常搜索推荐使用「均衡模式」，深度研究则启用「高质量模式」。

🧭 **自定义信息源** —— 支持搜索网页、社区讨论或学术论文等内容。更多数据源与集成正在持续开发中。

🧩 **小部件（Widgets）** —— 在相关场景下自动呈现的实用界面卡片，例如天气、计算、股票行情等快捷查询功能。

🔍 **基于 SearxNG 的网络搜索** —— 同时调用多个搜索引擎，全程保护您的身份隐私。对 Tavily 和 Exa 的支持即将上线，以提供更优结果。

📷 **图像与视频搜索** —— 在文本结果旁同步呈现视觉内容，搜索不再局限于文章。

📄 **文件上传** —— 可上传文档并就其内容提问。PDF、纯文本、图像等格式，Vane 均可理解。

🌐 **限定域名搜索** —— 当您明确知道目标网站时，可将搜索范围限制在特定站点内，特别适用于查阅技术文档或学术论文。

💡 **智能建议** —— 输入过程中实时提供精准搜索建议，助您构建更高效的查询语句。

📚 **发现（Discover）** —— 全天候浏览精选文章与热门内容，在不主动搜索的情况下保持信息更新。

🕒 **搜索历史** —— 所有搜索记录均本地保存，随时回溯您的探索历程，研究成果永不丢失。

✨ **更多功能即将上线** —— 我们正积极依据社区反馈开发新功能。欢迎加入我们的 Discord，共同塑造 Vane 的未来！

## 作为搜索引擎使用

若您希望将 Vane 用作 Google 或 Bing 等传统搜索引擎的替代方案，或希望在浏览器地址栏中为其添加快捷搜索入口，请按以下步骤操作：

1. 打开浏览器设置。
2. 进入「搜索引擎」设置项。
3. 添加一个新的网站搜索，URL 填写为：`http://localhost:3000/?q=%s`。若 Vane 并非本地运行，请将 `localhost` 替换为您的 IP 地址或域名，并将端口号 `3000` 替换为实际部署所用端口。
4. 点击添加按钮。此后，您即可直接在浏览器搜索栏中使用 Vane。

## 使用 Vane 的 API

Vane 同时面向开发者提供 API 接口，便于将这一强大搜索引擎集成至您自己的应用程序中。您可通过 API 执行搜索、调用多种模型，并获取结构化回答。

更多详情请参阅完整文档：[此处](https://github.com/ItzCrazyKns/Vane/tree/master/docs/API/SEARCH.md)。

## 将 Vane 暴露至局域网

Vane 基于 Next.js 构建，可直接处理全部 API 请求。默认情况下，它在同一局域网内即刻可用；即使配置了端口转发，亦可持续对外提供服务。
