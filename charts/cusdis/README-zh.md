## 与 Disqus 的对比

Cusdis 并不是为了成为 Disqus 的完全替代品而设计的。它的目标是为小型网站（例如你的静态博客）实现一个极简且可嵌入的评论系统。

以下是 Cusdis 的优缺点：

### 优点

- Cusdis 是开源且可自托管的，因此你可以完全掌控自己的数据。
- SDK 非常轻量（压缩后约 5KB）。
- 用户无需登录即可发表评论。
- Cusdis 完全不使用 Cookie。

### 缺点

- Cusdis 仍处于早期开发阶段。
- 没有垃圾评论过滤机制，因此你需要手动审核评论，评论在被批准之前不会显示。
- Disqus 是一家公司，而我们不是。
